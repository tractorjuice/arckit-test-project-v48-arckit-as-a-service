**Project ID**: 002

**Owner**: Mark Craddock (ArcKit as a Service Owner)
**Status**: Discovery

## Scope

This project specifies the **sovereign / on-premise / air-gapped deployment route** for ArcKit as a Service, deployable into customer-controlled environments — including UK Ministry of Defence and other sensitive sites — operating fully disconnected from vendor-controlled services. Architecturally anchored on Principle 21 in `projects/000-global/ARC-000-PRIN-v2.0.md`.

The managed multi-tenant SaaS route is covered by Project 001 (`projects/001-arckit-saas`). Both routes share one codebase; this project specifies the additional capability needed to package and accredit that codebase for sovereign use.
