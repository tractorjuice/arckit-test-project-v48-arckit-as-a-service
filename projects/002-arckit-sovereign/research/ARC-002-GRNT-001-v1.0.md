## Document Purpose

This document identifies and evaluates UK funding opportunities relevant to the ArcKit as a Service (Sovereign Deployment) project — a sovereign, air-gapped deployment of an enterprise architecture governance toolkit for UK MOD and accredited sensitive sites. Each opportunity is scored deterministically against the project profile using the `grants-uk-gov` rubric.

## Project Funding Profile

| Attribute | Value |
|-----------|-------|
| **Sector** | Defence (primary), Digital, AI |
| **Organisation Type** | SME |
| **TRL Level** | 4 (basis: ARC-002-REQ-v1.0 / ARC-002-SOBC-v1.0; alpha 2027-04-30, private beta 2027-08-31, GA 2027-12-31; currently at requirements/HLD with ADRs in place but no production code) |
| **Funding Sought** | £100,000 — £1,500,000 |
| **Project Timeline** | 2026-09-01 — GA 2027-12-31 |
| **Key Objectives** | Deliver sovereign air-gapped deployment for UK MOD and accredited sensitive sites; achieve MOD Secure by Design / JSP 440 / JSP 604 / NCSC CAF accreditation evidence per release; build HSM-backed signed-bundle supply-chain integrity (SBOM, hash inventory, signed manifests); deliver Long-Term Support release line with 24-month security patch SLAs; enable AI-assisted artefact generation via pluggable customer-controlled on-prem model endpoints. |
