## Overview

Companies House is the UK's authoritative register of companies. The Developer Hub publishes a REST API exposing companies, officers, filings, charges, insolvencies and persons-with-significant-control records under OGL-v3. In Project 002 it is a Lane B (vendor pre-sales) source for BR-006 SME-tier verification underpinning the cross-subsidy commercial model. It is **not** a runtime feed for the sovereign deployment.
